## 介绍

一个可以让你在 `template` 里渲染 `jsx` 或 `tsx` 组件的库，让你的 `ui` 渲染更灵活。

支持 `vue2` 和 `vue3`。

## 安装

for npm

```bash
npm i vue-xrender
```

for yarn

```bash
yarn add vue-xrender
```

for pnpm

```bash
pnpm add vue-xrender
```

## 浏览器

```html
<!-- Unpkg -->
<script src="https://unpkg.com/vue-xrender@latest" />

<!-- JsDelivr -->
<script src="https://cdn.jsdelivr.net/npm/vue-xrender@latest" />
```
